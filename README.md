# Murodjon Telegram Bot

Gemini AI bilan ishlaydigan shaxsiy Telegram yordamchi bot.

## Railway.app da deploy qilish (BEPUL)

### 1. GitHub ga yuklang
1. github.com ga kiring, yangi repo yarating (masalan: `murodjon-bot`)
2. Uchala faylni yuklang: `bot.py`, `requirements.txt`, `railway.toml`

### 2. Railway da ishga tushiring
1. railway.app ga kiring → "Start a New Project"
2. "Deploy from GitHub repo" → repongizni tanlang
3. Deploy tugadi — bot ishlaydi!

## Botni sozlash

`bot.py` faylida `SYSTEM_PROMPT` qismini o'zgartiring:
- Murodjon haqida ko'proq ma'lumot qo'shing
- Kompaniya, narxlar, mahsulotlar haqida ma'lumot kiriting
- Bot uslubini o'zgartiring

## Eslatma
- Telegram token va Gemini API keyni hech kimga bermang
- Railway bepul $5 kredit beradi (1 oyga yetadi)
